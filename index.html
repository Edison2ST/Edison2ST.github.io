CEA - Control en Almacenes

Deslice hacia abajo para ver más información

Estado del proyecto: Completo, privado

CEA es un software que solo se encuentra disponible en español y solamente yo estuve involucrado en este proyecto como desarrollador. Este software está hecho con PHP, MySQL, Javascript, HTML y Bootstrap, y está dirigido a usuarios que administran almacenes con varios dueños.

Hay muchos campos para los almacenes y para los artículos. En la sección de "Importar" (que importa datos de los almacenes y los artículos), hay un total de 25 campos que deben ser llenados utilizando un software Office como Microsoft Excel o LibreOffice Calc.

Este software encripta contraseñas de los usuarios y guarda esa encriptación en una base de datos MySQL. En términos generales, está muy bien diseñado para evitar la inyección SQL, XSS, LFI y RFI, entre otros.

El código fuente se encuentra en un repositorio privado.

EQN

Deslice hacia abajo para ver más información

Estado del proyecto: Completo y disponible para uso público

Este software resuelve sistemas de ecuaciones basadas en la eliminación de Gauss-Jordan, y muestra como salida la solución paso por paso con el resultado final. Está disponible en inglés (versión web y de escritorio) y en español (solo versión web).

La versión de escritorio fue desarrollada en C++, mientras que la versión web fue desarrollada en HTML, Javascript y Bootstrap (CSS). Puedes descargarla y utilizarla por tu propia cuenta sin necesidad de conexión a internet, o visitar la versión web y utilizarla sin necesidad de descargar nada.

Link de la versión web (online): https://edison2st.github.io/EQN/

Link del repositorio: https://github.com/Edison2ST/EQN
